このメッセージが表示されているということは、インストールが正常に完了していない可能性があります。

次の点を確認してください。

  • index.html ファイルを直接ウェブブラウザで開いているかもしれません。ウェブサーバー Apache を経由して表示してください。
  • .htaccess の設定が反映されていない可能性があります。Apache の設定で AllowOverride All などとして、設定の上書きを許可してください。
  • Apache モジュールの mod_rewrite が利用できない可能性があります。有効になるように設定する、またはモジュールのインストールを行ってください。

次のステップ

サイトマップを編集します

はじめに、サイト全体の構造をサイトマップに登録します。1行が1ページで表現されるCSV形式のファイルです。

サイトマップCSVは次のパスに置かれています。

./px-files/sitemaps/

テーマを編集します

テーマは、コンテンツエリア以外のHTMLソースを生成するPHPテンプレートです。

次のパスに置かれています。

./px-files/themes/

コンテンツを編集します

コンテンツは、コンテンツエリアのソースだけが記述されたシンプルなHTMLファイルです。
サイトマップの path 欄に登録したパスにHTMLファイルを作成してください。

あなたがいま見ているこのページは、テキストエディタで /index.html を開くとソースを確認できます。
このページも、削除や上書きしてあなたのサイトのトップページに置き換えてください。

パブリッシュします

サイトができあがったら、そのまま静的なHTMLに書き出すことができます。
次のコマンドで、パブリッシュを実行します。

$ php ./.px_execute.php /?PX=publish.run

しばらくすると、次のパスに、静的なHTMLファイルが生成されます。

./px-files/dist/
