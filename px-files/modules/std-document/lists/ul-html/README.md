リストを表現します。

ドラッグ可能なモジュール形式ではなく、 Markdown などのソースの形式で記述できます。

HTML や Markdown テキストの扱いに慣れているユーザーにとって記述しやすいインターフェイスで入力できます。

## Markdown での標準リスト記述例

```markdown
- List item 1
- List item 2
- List item 3
```

## HTML での標準リスト記述例

```html
<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
```

## 水平リスト

```html
<ul class="px2-horizontal-list">
	<li>水平リストアイテム1</li>
	<li>水平リストアイテム2</li>
	<li>水平リストアイテム3</li>
</ul>
```

## 注釈リスト

```html
<ul class="px2-note-list">
	<li>注釈リストアイテム1</li>
	<li>注釈リストアイテム2</li>
	<li>注釈リストアイテム3</li>
</ul>
```
